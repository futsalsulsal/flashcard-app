# フラッシュカード学習アプリ

テーマを問わず使える汎用フラッシュカード学習アプリ。

## ファイル構成

```
FlashcardApp/
├── flashcard.html      # アプリ本体
├── README.md           # このファイル
├── SKILL.md            # CSV生成スキル定義
└── data/               # CSVデータ格納フォルダ
    └── 経営用語集.csv   # サンプルデータ
```

## 使い方

1. `flashcard.html` をブラウザで開く
2. `data/` フォルダ内のCSVファイルをドラッグ&ドロップ or ファイル選択で読み込む
3. 「用語→意味」「意味→用語」の2モードで学習
4. 覚えた用語は「覚えた」ボタンでアーカイブ

## CSV形式

| 列名 | 必須 | 説明 |
|------|------|------|
| category | - | カテゴリ（あればフィルタ表示） |
| term | 必須 | 用語 |
| definition | 必須 | 意味・定義 |
| context | - | 使用文脈（あれば詳細表示） |

### CSVサンプル

```csv
category,term,definition,context
経営戦略,M&A,企業の合併・買収,成長戦略の文脈で使う
会計・財務,ROE,自己資本利益率,投資家が重視する指標
```

## 新しいテーマの追加

`/flashcard-csv-gen` スキルでCSVを自動生成できる。生成されたCSVは `data/` フォルダに格納される。

## 機能

- **用語→意味モード**: 用語を見て意味を思い出す
- **意味→用語モード**: 意味を見て用語を思い出す
- **アーカイブ**: 覚えた用語を一時退避。いつでも戻せる
- **カテゴリフィルタ**: ジャンル別に絞り込み
- **進捗バー**: 学習の進み具合を可視化
- **データ永続化**: ブラウザを閉じても進捗が保持される（localStorage）
- **汎用設計**: CSVを差し替えるだけで別テーマにも使える

## 動作環境

- モダンブラウザ（Chrome / Safari / Edge / Firefox）
- ローカルファイルとして開いても動作（`file://` 対応）
- サーバー不要
